# はじめての Cortex AI 関数

## 概要

Cortex AI 関数 は、SQL をマルチモーダルデータ向けの AI クエリ言語へと進化させ、強力な AI 機能を Snowflake の SQL エンジンに直接組み込みます。Google、OpenAI、Anthropic、Meta などの業界最先端の LLM を使用し、使い慣れた SQL コマンドでテキスト、画像、音声にわたるスケーラブルな AI パイプラインを構築できます。新しい FILE データ型によるマルチモーダルデータのネイティブサポートにより、Cortex AI 関数は AI_FILTER や AGGREGATE などの従来の SQL プリミティブと AI オペレーターをシームレスに統合し、エンタープライズグレードのセキュリティとガバナンスを維持しながら、アナリストが多様なデータ型をより効率的かつコスト効果の高い方法で処理できるようにします。
